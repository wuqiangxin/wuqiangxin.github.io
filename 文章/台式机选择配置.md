#### 2. 主板
1. 主板 和 CPU 有兼容性。 选择主板前，要先选好 CPU
2. 可以在 购物网站上直接搜索 CPU型号套装  例如： Intel 酷睿i5 1135G7主板套装

</br>

#### 3. 内存条
1. 4GB、 8GB、 16GB、 32GB 通常我们选择 8GB 和 16GB 是足够的
2. 选择 内存条并不是越大越好 

</br>

#### 4. 显卡 （画质、 游戏）
1. 独立显卡： 贵、性能好、发热量高
2. 集成显卡： 便宜些、性能一般、发热量低
